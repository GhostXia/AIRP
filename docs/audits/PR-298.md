### 4.3 安全边界判断

经 F3 / F9 / F12 / F13 / F14 + NbF / NbU 修复后，计划在以下安全边界与仓库硬约束对齐：

- **CSP `unsafe-eval` 禁止**：harness 不用 `new Function` / `eval`，waitFor 改为 predicateId 注册表（F3/F9/I1）。
