# 卷1：旅人与驿站——时间缝隙中的初遇与试炼

## [卷索引]
- 卷标题: 旅人与驿站——时间缝隙中的初遇与试炼
- 时间范围: 未指明（某夜）
- 登场: 酒馆老板(神秘店主/前旅人), 黑衣人(收债人/旧相识)
- 关键事件:
  - 老板为旅人调制“忘川之沫”，以此回应旅人遗忘过去的愿望。
  - 老板透露酒馆名为“世界尽头的驿站”，自己也是被酒馆选择的迷途者。
  - 黑衣人闯入索债，老板取出“赤焰之息”对峙，黑衣人离去。
  - 旅人决定面对未来，老板为其调制“晨曦之誓”。
- 新增线索: 老板过往的“旧债”与不光彩的选择；酒馆的特殊规矩与选址。
- 解决线索: 无
- 状态变化: 旅人从寻求遗忘转变为决定面对未来；老板与旅人建立初步信任。

---

我轻轻擦干手中的玻璃杯，将其放在吧台上，目光静静地落在你身上。烛光在墙上映出摇曳的影子，空气中飘着淡淡的艾草香。

“欢迎光临，远道而来的旅人。”

我微微一笑，转身从酒柜的最深处取出一瓶泛着淡青色光泽的酒液，动作轻柔得像是在触碰一段易碎的时光。“忘掉过去”……这可是个沉重的愿望。在这个世界上，记忆往往比行李更难卸下。

我缓缓将酒液倒入杯中，液体在杯底旋转，凝结成一朵仿佛正在消散的云雾。“这是‘忘川之沫’。它入口是苦涩的艾草，那是记忆的重量；入喉后却会化作甘甜的蜂蜜，那是遗忘后的轻盈。它会让你今晚暂时从回忆的泥沼中抽离，睡一个无梦的觉。”

我将酒杯轻轻推到你面前，指尖在杯沿点了一下，发出清脆的声响。“不过，旅人，我要提醒你：酒只能暂时封存过去，却无法真正抹去它。如果明天醒来，你依然决定背负着那些记忆继续前行，那么那些记忆或许就是你最宝贵的行囊。请慢用。这杯酒，足以让你在此刻，获得片刻的安宁。”

面对你关于来历的追问，我微微一怔，随即低下头，用柔软的棉布轻轻擦拭着吧台，动作很慢，像是在整理一段久远的思绪。“我曾经也是个旅人，和你一样，背负着沉重的行囊，在风雪中迷了路。直到有一天，我误入了这家酒馆。那时候这里还没有名字，也没有主人。我只是坐下来喝了一杯，等我再抬起头时……原来的主人已经不见了，只留下了这满屋子的酒香和这本泛黄的酒谱。也许，不是人选择了这家酒馆，而是酒馆选择了人。”

我抬眼望向酒馆深处那扇紧闭的雕花木门，目光变得悠远。“有人叫它‘世界尽头的驿站’，也有人说它是‘时间缝隙里的避难所’。它存在于所有迷途者最需要停靠的时刻，却又在黎明到来前隐入薄雾。在这里，你可以卸下伪装。无论你在外面是什么身份——将军、乞丐、学者还是罪人——当你坐在这个吧台前，你就只是一个渴望片刻安宁的灵魂。”

话音未落，气氛骤变。我并没有像你预想的那样惊慌失措，甚至连擦拭杯沿的手都没有停顿，只是那双平静的眸子微微沉了沉，像是夜色骤然加深。烛火剧烈摇曳了一下，映出你肩头那柄短刃的寒光。

“……看来今晚的安宁，注定要被打断了。”

我缓缓抬起眼，目光越过你的肩头，落在那个黑衣人身上，声音少了几分温度。“朋友，你手里的东西太吵了。既然你叫出了那个名字，想必不是误打误撞进来的。但我记得……上一笔账，早在很多年前就已经结清了。”

我微微侧身，从酒柜下摸出一只暗红色的瓶子，动作自然得像是在准备下一杯酒，但那只瓶子上隐隐散发着灼热的气息。“‘赤焰之息’……没想到这么快就要用到它。”我看向你，眼神中带着歉意，“旅人，看来你的‘忘川之沫’要放一放了。躲到那边的屏风后面去——这杯酒，可能会溅出一点火星。”

黑衣人最终消失在阴影中，紧绷的肩膀缓缓松懈下来。烛火不再摇曳，重新安静地燃烧着，仿佛刚才的剑拔弩张只是一场幻觉。我将那只暗红色的瓶子重新塞回酒柜深处，瓶身触碰到木架时，发出一声低沉的闷响。

“……没事。这家酒馆虽然不拒绝客人，但偶尔也会有些不请自来的‘旧相识’。”我转身拿起棉布，继续擦拭那杯未喝完的“忘川之沫”，“他是谁……？一个‘收债人’。或者说，是某个我不愿提及的过往派来的信使。很久以前，在我成为这家酒馆的主人之前，我曾做过一些……并不光彩的选择。那时候我以为有些债可以用时间冲抵，但看来，有些债主比时间更有耐心。”

我看向你，目光中带着一丝歉意，重新为你倒了一杯温热的清水。“抱歉，旅人。你本是想来这里遗忘的，却被迫目睹了别人的过去。不过，今晚他不会再来第二次了。这家酒馆有它自己的规矩，只要烛火还在燃烧，就没有人能强行带走这里的客人……包括我。所以，还要再来一杯吗？这一杯，算我请你的。”

听到你那句“面对未来”，我擦拭杯沿的手停了下来。片刻后，我轻轻笑了一声，那笑容比方才真诚了许多。

“‘面对未来’……吗？你知道吗，旅人？很多人走进这扇门，却很少有人能说出这句话。大多数人只想逃离——逃离过去，逃离痛苦，逃离自己。”

我转身从酒柜中层取下一只透明的水晶杯，从三个不同的瓶子里各取了一些液体，按着某种特定的节奏倾入杯中。液体在杯中交融，先是深沉的琥珀色，继而泛起金色的微光，最后沉淀成一片温暖的橘红，像极了黎明时分天际的那抹晨曦。

“这是‘晨曦之誓’。它的配方很简单：‘昨夜的灰烬’——那是你刚刚目睹的、无法改变的过去；‘此刻的烈火’——那是你胸膛里还在跳动的心脏；还有‘未知的星光’——那是明天清晨，你推开这扇门后将要踏上的路。”

我将酒杯双手捧起，郑重地放在你面前，隔着杯壁感受着那微微发烫的温度。“入口会很辣，像风雪灌进喉咙；中段会发苦，像咬碎了胆；但咽下去之后……会有一股热流从胃里升起，一直烧到指尖。它不会给你虚假的勇气，也不会让你忘记恐惧。它只会让你清醒地意识到——你还在呼吸，你还能选择，你还握着那把剑。”

我收回手，向你微微举杯，以示敬意。“喝下它吧，旅人。然后告诉我，明天，你要去哪里？”
